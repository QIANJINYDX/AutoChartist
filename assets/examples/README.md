# 示例数据文件说明

本目录包含用于测试 AutoChartist 的示例数据文件。

## 文件列表

### 1. `sales_data.csv`
简单的销售数据示例，包含：
- `date`: 日期（时间序列）
- `sales`: 销售额（数值）
- `region`: 地区（分类）
- `product`: 产品（分类）

**适用场景**：
- 时间序列折线图
- 按地区分组的条形图
- 产品销售对比

### 2. `student_scores.csv`
学生成绩数据，包含：
- `student_id`: 学生ID
- `name`: 姓名
- `age`: 年龄（数值）
- `math_score`: 数学成绩（数值）
- `english_score`: 英语成绩（数值）
- `science_score`: 科学成绩（数值）
- `class`: 班级（分类）

**适用场景**：
- 散点图（身高 vs 体重）
- 箱线图（成绩分布）
- 分组条形图（各科目平均分）
- 相关系数热力图

### 3. `test_data.csv` ⭐ 推荐
综合测试数据，包含多种数据类型：
- `date`: 日期（时间序列，2024年1-3月）
- `product`: 产品名称（分类）
- `category`: 产品类别（分类：Electronics, Clothing, Food）
- `sales`: 销售额（数值）
- `quantity`: 数量（数值）
- `region`: 地区（分类：North, South, East, West）
- `rating`: 评分（数值，3.7-4.9）
- `cost`: 成本（数值）

**适用场景**：
- ✅ 时间序列图（按日期展示销售额趋势）
- ✅ 分组条形图（按类别/地区对比销售额）
- ✅ 散点图（销售额 vs 数量，或销售额 vs 成本）
- ✅ 堆叠条形图（按类别堆叠展示）
- ✅ 箱线图（不同地区的销售额分布）
- ✅ 热力图（相关系数矩阵）
- ✅ 带误差线的图表

## 使用建议

1. **首次测试**：使用 `test_data.csv`，它包含最丰富的数据类型
2. **简单测试**：使用 `sales_data.csv`，数据量小，加载快
3. **教育场景**：使用 `student_scores.csv`，适合展示学生成绩分析

## 示例提示词

使用 `test_data.csv` 时，可以尝试以下提示词：

- "画一个按日期展示销售额的折线图"
- "展示各产品类别的销售额对比，使用条形图"
- "绘制销售额和数量的散点图，用颜色区分地区"
- "展示各地区的销售额分布，使用箱线图"
- "画一个按月份聚合的销售额折线图，加上95%置信区间"
- "展示产品类别和地区的销售额热力图"
- "绘制销售额和成本的散点图，添加回归拟合线"
